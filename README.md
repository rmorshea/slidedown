```

Create an awesome slide deck.

Present it to awesome people.

Profit?
```

# How?

1. Install `slidedown` with `pip`

```bash
pip install slidedown
```

2. Start presenting your markdown files

```bash
slidedown README.md
```

*try it out with this README file!*
